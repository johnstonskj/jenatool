Creating SPARQL Queries

Container
the workspace container in which the resource will be created.
File name
the name of the workspace resource to create.
Query Form
the form of the SPARQL query to create, will select a template.

Running Queries

Query Resource
the workspace resource containing the SPARQL query.
Target Dataset
the dataset to execute the query against.
Format
the format of the output of DESCRIBE and CONSTRUCT queries.
Query Context
the model or models over which the query will execute.
